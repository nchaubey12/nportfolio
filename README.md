# ✦ Nidhi Chaubey — Portfolio

A dual-experience personal portfolio with a gateway landing page that routes visitors into two distinct views based on who they are.

---

## 🗂 Structure

```
portfolio/
├── index.html                  # Gateway — choose your experience
├── portfolio_recruiter.html    # Professional view (recruiters)
├── portfolio_friend.html       # Friendly view (connections & curious minds)
└── images/
    ├── image_.png               # Profile photo
    ├── visiontutor_screenshot.png
    └── smartcity_screenshot.png
```

---

## ✨ Features

### Gateway (`index.html`)
- Full-screen white professional theme by default
- Hover over **Recruiter** card → stays clean white with blue accents
- Hover over **Friend** card → whole page shifts to deep dark purple with glowing particles
- EN / DE language toggle (top right)
- Smooth fade transition between pages

### Recruiter Portfolio (`portfolio_recruiter.html`)
- Clean corporate aesthetic — IBM Plex Sans + Playfair Display
- White/dark mode toggle (defaults to dark)
- EN / DE language switch — full translation of all content
- Sections: Hero · Work Experience (vertical timeline) · Projects · Education · Certifications · Contact
- Real project screenshots for VisionTutor AI and Smart City Platform
- Availability pills: Working Student (20h/week) + Open for Mandatory Internship from July

### Friendly Portfolio (`portfolio_friend.html`)
- Blue-steel dark theme with animated magic dust particles
- Floating tech labels + orbiting portrait rings
- Animated code snippets in hero
- EN / DE language switch
- Sections: Hero · About · Experience · Projects (Spellbook) · Education · Certifications · Contact
- Warm, human contact copy

## 🛠 Tech Stack

Pure HTML · CSS · Vanilla JavaScript — no frameworks, no build tools, no dependencies.

Fonts via Google Fonts:
- `DM Serif Display` · `DM Sans` · `Space Mono` (friendly view)
- `Playfair Display` · `IBM Plex Sans` · `IBM Plex Mono` (recruiter view)

## 🚀 Deploy

Drop all files into any static host (GitHub Pages, Netlify, Vercel):

```
index.html
portfolio_recruiter.html
portfolio_friend.html
images/
  image_.png
  visiontutor_screenshot.png
  smartcity_screenshot.png
```

> All image paths are relative — keep `images/` in the same directory as the HTML files.

### GitHub Pages (quickest)
1. Push to a repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/portfolio` folder
4. Visit `https://<username>.github.io/<repo>/portfolio/`

---

## 🌐 Language Support

Both portfolios support **English** and **German** via a built-in toggle. All text uses `data-en` / `data-de` attributes — no external i18n library needed.
